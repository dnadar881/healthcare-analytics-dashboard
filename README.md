# 🏥 Healthcare Analytics Dashboard

## 📊 Project Overview
This project is an end-to-end healthcare analytics solution built using SQL and Power BI. It focuses on analyzing patient data, hospital performance, and revenue insights to support data-driven decision-making.

---

## 🚀 Key Features

- KPI tracking (Total Patients, Revenue, Avg Billing, Avg Stay)
- Admission type analysis (Emergency, Urgent, Elective)
- Time-series trend analysis (Monthly & Yearly)
- Top-performing hospitals by revenue
- Interactive filters (Gender, Medical Condition, Hospital)
- AI-style dynamic insights using DAX

---

## 🛠️ Tools & Technologies

- SQL (MySQL)
- Power BI
- DAX
- Excel (Data Cleaning)

---

## 📂 Dataset

The dataset includes:

- Patient demographics
- Admission & discharge dates
- Medical conditions
- Billing information
- Hospital details

---

## 📈 Dashboard Insights

- Admission distribution is balanced (~33% each category)
- Total revenue exceeds $1B
- Average patient stay is ~15 days
- Emergency admissions contribute significantly to hospital load
- No strong seasonal spikes observed in admissions

---

## 🤖 AI Feature

A dynamic insight system was implemented using DAX:

- Automatically updates insights based on filters
- Simulates AI-driven decision support
- Highlights emergency admission trends
